# Inspiration Galleries

Before locking scenes and motion, pull current references so the build aims at a
premium, award-tier bar — not a generic template. The deliverable should read like
a site that costs **$5,000+**: a signature moment, real craft in motion, and
restraint. Browse these, name the 2–4 specific references you're borrowing from,
and say *what* you're taking (a transition, a layout idea, a material) — never copy
wholesale.

## Sources

- **Awwwards — 3D** — https://www.awwwards.com/websites/3d/
  The bar for craft. Study award-winners for hero moments, scroll choreography,
  and how the best sites earn a "wow" without sacrificing usability.
- **Landing.love — 3D websites** — https://www.landing.love/categories/3d-website/
  Conversion-minded 3D landing pages. Good for pairing a 3D hero with a real
  funnel (the thing that justifies the budget to a client).
- **Webflow — Made in Webflow (3D animation, cloneable)** —
  https://webflow.com/made-in-webflow/3danimation?cloneable=true
  Cloneable builds — useful to see *how* an effect is actually assembled and to
  estimate effort, not just admire the result.
- **Vev — Showcase** — https://www.vev.design/showcase/
  Editorial + interactive case studies; strong for scroll-driven storytelling and
  mixing rich media with 3D.
- **Refs.gallery — 3D tag** — https://www.refs.gallery/tags/3d
  Broad, fast-moving reference wall for quick visual benchmarking against the
  client's industry.

## How to use them in the plan

1. Match the client's industry/positioning to 2–4 specific references (link them).
2. For each, note the one idea you're adapting (e.g. "Awwwards site X's
   material-reveal on scroll → use for the product hero").
3. Sanity-check ambition against budget: an award-tier signature moment is worth
   the spend; ten of them is scope creep. Pick **one** signature, support it with
   restrained motion elsewhere.
4. Translate references into the **Signature hero concept** and **Interactions &
   motion** sections — don't leave them as a mood board.

> Inspiration drives *form and motion*, never *content*. On an existing site the
> client's real copy, products, and assets are fixed (see the content-fidelity
> contract in SKILL.md); references only inform how that content is staged and
> animated.
