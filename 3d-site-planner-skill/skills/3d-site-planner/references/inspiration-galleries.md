# Inspiration Galleries & Exemplars

Before locking scenes and motion, pull current references so the build aims at a
premium, award-tier bar — not a generic template. The deliverable should read like
a site that costs **$5,000+**: one signature moment, genuine motion craft, and
restraint. Browse these, name the 2–4 specific references you're borrowing from, and
say *what* you're taking (a transition, a layout idea, a material) — never copy
wholesale.

> The north-star bar for this skill is **igloo.inc** (https://www.igloo.inc/) —
> Awwwards Site of the Day + Developer Award. See `north-star-igloo.md` for the full
> deconstruction. The point of igloo isn't "more 3D" — it's **one coherent metaphor
> executed end-to-end**, UI and 3D in a single WebGL context, procedural systems that
> scale, and obsessive load/shader-compile management so the heaviness is invisible.
> That gap — concept + craft + performance — is what separates an award-tier build
> from a Spline embed.

## How to use them in the plan

1. Match the client's industry/positioning to 2–4 specific references (link them).
2. For each, note the one idea you're adapting (e.g. "igloo's frost/dissolve scene
   transition → use between the product and process sections").
3. Sanity-check ambition against budget: an award-tier signature moment is worth the
   spend; ten of them is scope creep. Pick **one** signature, support it with
   restrained motion elsewhere.
4. Translate references into the **Signature hero concept**, **Motion & animation
   system**, and **Interactions & motion** sections — don't leave them as a mood board.

> Inspiration drives *form and motion*, never *content*. On an existing site the
> client's real copy, products, and assets are fixed (see the content-fidelity
> contract in SKILL.md); references only inform how that content is staged and animated.
