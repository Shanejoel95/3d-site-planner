# Analysis Rubric

Score the client to make the analysis consistent and to derive the highlights /
drawbacks lists objectively. Rate each dimension **1–5** (1 = weak/absent,
5 = strong/standout) with a one-line justification grounded in a real finding.

## Dimensions

1. **Digital presence** — does a site exist, and how capable is it?
2. **Brand assets** — logo, colors, typography, consistency, distinctiveness.
3. **Photography / visual content** — quality, quantity, 3D-readiness.
4. **Reputation** — review rating, volume, sentiment, press/awards.
5. **Content readiness** — depth and quality of copy, testimonials, story.
6. **Video / motion assets** — usable footage and production quality.
7. **Audience clarity** — how well-defined the target customer is.
8. **Competitive differentiation** — room to stand out (esp. via 3D).
9. **3D-fit** — does the offering visually benefit from 3D (products, spaces,
   process) vs. being purely informational?
10. **Conversion clarity** — is the primary goal (call/booking/purchase/lead) clear?

## Deriving the two lists

**Highlights** = dimensions scoring 4–5, restated as assets to amplify in the build.
- Example: Reputation = 5 (4.8★, 200+ reviews) → "Make verified social proof a hero
  element."
- Example: Photography = 4 (clean product shots) → "3D product showcase using
  existing imagery."

**Drawbacks / gaps** = dimensions scoring 1–2, restated as actions the plan must take.
- Example: Photography = 1 (no usable images) → "Budget a product shoot or generate
  3D/AI assets before build."
- Example: Content readiness = 2 (thin copy) → "Messaging workshop; copy is a
  deliverable."

Every drawback must map to a concrete item later in the build brief or roadmap. A
listed-but-unaddressed drawback is a planning error.

## Reading the overall shape
- **Many highlights, few gaps** → an *amplification* project; lean into a signature
  custom moment.
- **Few highlights, many gaps** → a *foundation* project; budget brand/content/photo
  work before heavy 3D; consider 3D-lite or Spline to control cost.
- **Strong 3D-fit + weak competitors** → 3D is the differentiator; justify investment.
- **Low 3D-fit (purely informational)** → keep 3D restrained; don't force it.

Include the scored table in the plan's analysis section so the recommendation is
traceable to evidence.
